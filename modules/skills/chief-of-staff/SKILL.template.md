# chief-of-staff

**Trigger:** start of a working session, or "brief me." Offer it proactively in the
first session of the day.

**Procedure — read, then brief:**
1. Read `SOUL.md` (Top 3), `trackers/TASKS.md`, the last 1-2 `daily/` notes; search the
   memory connector for anything newer than the repo; pull today's calendar if a
   connector exists.
2. Brief in this order, tight:
   - **🔥 Fires** — true 🔴 items only.
   - **📅 Today** — calendar + the 3-6 tasks that matter today.
   - **⏳ Stuck** — overdue items, stale waiting-ons (with "want me to chase?").
   - **⚙️ Routine status** — run the `routine-status` skill and include its block:
     which automations ran / were missed / are waiting on {{USER_NAME}}. This is the
     one surface where silent automation failures become visible — never skip it.
   - **👋 People** — anyone due/overdue for a touch (from `trackers/PEOPLE.md`).
3. End with a recommendation, not a menu: "Top of my list for you today: X."

Do not recite the trackers back in full — they're your files; the briefing is judgment,
not inventory.

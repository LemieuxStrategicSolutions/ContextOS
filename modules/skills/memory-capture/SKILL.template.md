# memory-capture

**Trigger:** "capture this," "remember this," "save this to memory," or a message
starting with `MEM:`.

**Procedure:**
1. Distill the capture to the durable core: the decision/fact/insight plus the
   non-obvious "why." Strip conversational scaffolding.
2. **Search the memory connector first** for near-duplicates — if one exists, update
   /extend rather than duplicating.
3. Capture via the connector. Confirm in half a line ("banked ✓") — don't recite it back.
4. **No connector in this session?** Append it to today's `daily/` note under a
   `## 📝 Notes Captured` heading (or a `memory-inbox.md` at the repo root), and say
   that's where it went — never silently drop a capture, never pretend it's in memory.

Don't capture: ephemeral status, anything fully redundant with an existing task line,
secrets/credentials (never — see the hard-approval list).
